Links en html

El atributo "target" para la etiqueta "a" se usa para definir el como se abrira el enlace

  • target="_self" : Defecto. Abre el documento en la misma ventana/pestaña en la que se hizo clic
  • target="_blank" : Abre el documento en una nueva ventana o pestaña
  • target="_parent" : Abre el documento en el marco principal
  • target="_top" : Abre el documento en el cuerpo completo de la ventana

Colores de enlace html 
                Por defecto, aparecerá un enlace como este (en todos los navegadores):
                    Un enlace no visitado está subrayado y azul
                    Un enlace visitado está subrayado y morado
                    Un enlace activo está subrayado y rojo.
                Puede cambiar los colores del estado del enlace usando CSS
